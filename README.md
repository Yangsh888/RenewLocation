# RenewLocation - 面向 TypeRenew 的轻量评论 UA 与归属地插件
本插件是一款面向 TypeRenew 的轻量评论增强插件，适配 PHP 8 与 MySQL 8，围绕评论区的浏览器识别、操作系统识别、IP 归属地展示与纯真 CZDB 数据库自动更新，提供一套默认可用、便于统一管理的前台展示能力。

## 功能特性
- **评论 UA 信息展示**
  - 浏览器识别
  - 操作系统识别
  - 保持轻量输出，不展示冗长版本号
  - 对常见脚本、爬虫环境提供基础识别
- **IP 归属地展示**
  - 支持 IPv4 归属地查询
  - 支持 IPv6 归属地查询
  - 支持纯真 CZDB v4 / v6 数据库
  - 支持归属地轻过滤与长文本省略显示
  - 支持 tooltip 查看完整归属地信息
- **前台评论区轻量注入**
  - 默认/兼容模板自动注入评论元信息
  - 第三方模板支持手动调用输出
  - 浏览器、系统、归属地标签顺序统一
  - 归属地解析失败时可回退显示脱敏 IP
- **数据库自动更新**
  - 支持后台配置纯真 `czdb.zip` HTTPS 下载地址
  - 支持按配置天数自动检查更新
  - 下载后自动解压并替换 v4 / v6 数据库
  - 数据库文件仅存放在插件目录内，不污染站点根目录
- **轻量配置与兼容性处理**
  - 直接使用普通插件配置页，不引入独立控制面板
  - 支持前台显示项独立开关
  - 支持第三方模板调用示例直接复制
  - 保持 TypeRenew 轻量、简洁的内核风格

## 适用场景
- 基于 TypeRenew 构建的个人博客、内容站、小型资讯站
- 希望在评论区展示浏览器、系统和 IP 归属地信息的站点
- 希望使用纯真 CZDB 数据库并需要评论信息增强能力的主题或站点

## 安装方式
1. 将插件目录上传到站点的 `/usr/plugins/` 目录
2. 确保目录名称为 `RenewLocation`
3. 登录后台，进入“控制台 -> 插件”
4. 找到并启用 `RenewLocation`
5. 进入插件配置页，填写纯真数据库下载地址、访问密钥与更新周期

## 配置说明
插件配置页主要包含以下模块：
- **插件状态**
  - 启用或停用插件
- **前台显示项**
  - 显示浏览器
  - 显示操作系统
  - 显示归属地
  - 显示脱敏 IP 兜底
- **第三方模板调用**
  - 提供评论区手动调用示例代码
  - 适用于未兼容自动注入结构的主题
- **自动更新**
  - 自动更新开关
  - 纯真数据库下载地址
  - 访问密钥
  - 更新周期
  - 请求超时

## 第三方模板调用示例
若主题评论结构与默认/兼容模板不同，可在评论循环中手动放置以下代码：

```php
<?php if (class_exists('\TypechoPlugin\RenewLocation\View')) { \TypechoPlugin\RenewLocation\View::renderComment($comments ?? $this); } ?>
```

建议调用位置：
- 评论时间后
- 回复按钮前

## 数据库更新说明
- 插件使用纯真提供的 `czdb.zip` 数据包
- 压缩包内应包含：
  - `cz88_public_v4.czdb`
  - `cz88_public_v6.czdb`
- 更新时会先下载到插件数据目录的临时 staging 目录
- 校验所需文件齐全后再替换正式数据库文件
- 更新失败时会自动回滚，避免留下半更新状态

## 目录结构
```text
RenewLocation/
├── lib/                 # 内置 CZDB 查询与解包支持
│   └── Czdb/
├── data/                # 纯真数据库文件存放目录
├── Plugin.php           # 插件主入口与 Hook 注册
├── Settings.php         # 配置默认值、归一化与读写
├── Action.php           # 自动更新与异步入口
├── View.php             # 前台样式、标签渲染与自动注入
├── Ua.php               # 浏览器与系统识别
├── Ip.php               # IP 归属地查询
├── Loader.php           # CZDB 类自动加载
├── Text.php             # 文本清洗与脱敏辅助
└── readmd.md            # 插件说明文档
```

## 注意事项
- 插件不会修改 TypeRenew 核心类，所有能力均通过插件机制接入
- 数据库下载地址当前仅接受 HTTPS 链接
- 若主题评论结构差异较大，自动注入可能无法命中，请改用手动调用方式
- 归属地标签会优先保证布局稳定，超长内容会自动省略，并在悬浮时显示完整信息
- 若浏览器或系统标签全部关闭，插件会自动跳过对应前台处理链路，减少无效开销
- 使用前请确保服务器具备 `ZipArchive` 扩展与正常的外部 HTTPS 访问能力
